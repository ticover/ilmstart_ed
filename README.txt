IlmStart EDU — учебный центр как e-kundalik / LMS

Что есть в проекте:
1. Вход по логину и паролю.
2. 3 роли: admin, teacher, student.
3. Админ-панель:
   - ученики
   - учителя
   - предметы
   - группы
   - расписание
   - месячные оплаты
   - новости
   - объявления
   - просмотр результатов
4. Учитель:
   - свои группы
   - перекличка
   - результаты учеников
   - пройденные темы
   - тесты и вопросы
5. Ученик:
   - мои предметы
   - расписание
   - результаты
   - месячные оплаты
   - тесты
   - уведомления
6. Адаптивный интерфейс под телефон и компьютер.
7. PWA-заготовка: manifest.json и service-worker.js.

Как запустить:
1. Открой папку IlmStart_EDU в PyCharm или VS Code.
2. Открой терминал в этой папке.
3. Создай виртуальное окружение:
   python -m venv venv
4. Активируй его:
   Windows:
   venv\Scripts\activate

   macOS/Linux:
   source venv/bin/activate
5. Установи библиотеки:
   pip install -r requirements.txt
6. Запусти проект:
   python app.py
7. Открой в браузере:
   http://127.0.0.1:5000

Запуск на телефоне через Wi-Fi:
1. Ноутбук и телефон подключи к одному Wi-Fi.
2. На ноутбуке запусти start_wifi.bat или команду:
   python app.py
3. Узнай IP ноутбука командой:
   ipconfig
4. Найди IPv4 Address, например 192.168.1.8.
5. На телефоне открой:
   http://192.168.1.8:5000
6. Если не открывается, разреши Python/Flask в Windows Firewall для частной сети.
